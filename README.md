# MINIPROYECTO_SENA 
PROYECTO EN GRUPO PARA  EL SENA, MAXIMO 15 DIAS para practicar y afianzar los conocimientos en POO. El objetivo del software es facilitar la administracion de habitaciones, reservas, clientes y pagos utilizando unicamente la consola para la visualizacion. El codigo se dividió en 6 archivos, 2 por cada integrante del equipo.
modulos:
base de datos: rea y gestiona la conexion a la base de datos SQlite y todas las tablas necesarias
clientes: registra, actualiza y consulta la informacion de los clientes
habitaciones: permite crear, modificar, eliminar habitaciones asi como cambiar su estado
reservas: crea, modifica y cancela reservas controlando disponibilidad y fechas
pagos: simula la validacion y confirmacion asincronica de pagos
main: reune todos los modulos y presenta un menu interactivo en consola

requisitos: python 3.10 o superior, SQlite
